Linguagens de Consulta e Manipulação de Dados no SQL

O SQL (Structured Query Language) é a linguagem padrão para interação com bancos de dados relacionais. Ele é dividido em várias sub-linguagens, cada uma com um propósito específico. As principais são:

  1. Linguagem de Definição de Dados (DDL)

  2. Linguagem de Manipulação de Dados (DML)

  3. Linguagem de Consulta Estruturada (SQL)

1. Linguagem de Definição de Dados (DDL)

A DDL é responsável por definir e modificar a estrutura de um banco de dados. Ela é usada para criar, alterar e excluir objetos de banco de dados, como tabelas, índices, visões e esquemas.

Principais comandos DDL:

  • CREATE: Cria objetos como tabelas, índices, views.

  • ALTER: Modifica a estrutura de objetos existentes.

  • DROP: Exclui objetos existentes.

  • TRUNCATE: Remove todos os dados de uma tabela sem remover a estrutura da tabela.

Exemplo prático:

  1. Criando uma tabela
    O comando CREATE TABLE é usado para criar uma nova tabela no banco de dados.

    sql
Copiar código
CREATE TABLE Clientes (
    id_cliente INT PRIMARY KEY,
    nome VARCHAR(100),
    email VARCHAR(100),
    data_nascimento DATE
);

    Explicação: Cria uma tabela chamada Clientes com quatro colunas: id_cliente, nome, email e data_nascimento.

  2. Alterando uma tabela O comando ALTER TABLE é usado para modificar a estrutura de uma tabela existente.

    sql
Copiar código
ALTER TABLE Clientes ADD telefone VARCHAR(15);

    Explicação: Adiciona uma nova coluna chamada telefone à tabela Clientes.

  3. Excluindo uma tabela O comando DROP TABLE é usado para excluir uma tabela e seus dados.

    sql
Copiar código
DROP TABLE Clientes;

    Explicação: Exclui a tabela Clientes do banco de dados.

2. Linguagem de Manipulação de Dados (DML)

A DML é responsável por manipular os dados dentro das tabelas de um banco de dados. Ela permite inserir, atualizar, excluir e consultar os dados armazenados.

Principais comandos DML:

  • INSERT: Insere dados em uma tabela.

  • UPDATE: Atualiza dados existentes em uma tabela.

  • DELETE: Exclui dados de uma tabela.

  • SELECT: Recupera dados de uma tabela.

Exemplo prático:

  1. Inserindo dados O comando INSERT INTO é usado para inserir dados em uma tabela.

    sql
Copiar código
INSERT INTO Clientes (id_cliente, nome, email, data_nascimento)
VALUES (1, 'João Silva', 'joao@exemplo.com', '1985-06-15');

    Explicação: Insere um novo registro na tabela Clientes com valores para as colunas id_cliente, nome, email e data_nascimento.

  2. Atualizando dados O comando UPDATE é usado para atualizar registros existentes em uma tabela.

    sql
Copiar código
UPDATE Clientes
SET email = 'joao.silva@novoemail.com'
WHERE id_cliente = 1;

    Explicação: Atualiza o campo email da tabela Clientes para um novo valor, onde o id_cliente é 1.

  3. Excluindo dados O comando DELETE é usado para excluir registros de uma tabela.

    sql
Copiar código
DELETE FROM Clientes
WHERE id_cliente = 1;

    Explicação: Exclui o registro da tabela Clientes onde o id_cliente é igual a 1.

  4. Consultando dados O comando SELECT é usado para recuperar dados de uma tabela.

    sql
Copiar código
SELECT nome, email
FROM Clientes
WHERE data_nascimento < '1990-01-01';

    Explicação: Recupera os nomes e emails de todos os clientes cuja data_nascimento seja anterior a 1º de janeiro de 1990.

3. Linguagem de Consulta Estruturada (SQL)

A SQL é uma linguagem de consulta usada para recuperar e manipular dados em bancos de dados relacionais. Embora "SQL" seja frequentemente usado para se referir a todas as operações de consulta e manipulação de dados, o termo Linguagem de Consulta Estruturada é usado especificamente para consultas de leitura e análise dos dados.

Principais comandos de SQL:

  • SELECT: Usado para consultar e selecionar dados de uma tabela.

  • JOIN: Usado para combinar registros de duas ou mais tabelas.

  • GROUP BY: Usado para agrupar registros com base em um critério específico.

  • ORDER BY: Usado para ordenar os resultados de uma consulta.

Exemplo prático:

  1. Consultando todos os dados de uma tabela

    sql
Copiar código
SELECT * FROM Clientes;

    Explicação: Recupera todos os dados da tabela Clientes. O asterisco (*) indica que todas as colunas devem ser retornadas.

  2. Consultando com filtros

    sql
Copiar código
SELECT nome, email
FROM Clientes
WHERE nome LIKE 'João%';

    Explicação: Recupera os nomes e emails de todos os clientes cujo nome começa com "João".

  3. Usando JOIN para combinar tabelas Suponha que você tenha uma segunda tabela chamada Pedidos. Você pode usar JOIN para combinar dados de ambas as tabelas.

    sql
Copiar código
SELECT c.nome, p.id_pedido, p.valor
FROM Clientes c
JOIN Pedidos p ON c.id_cliente = p.id_cliente;

    Explicação: Recupera os nomes dos clientes, juntamente com o id_pedido e valor dos pedidos, fazendo uma junção entre as tabelas Clientes e Pedidos com base na coluna id_cliente.

  4. Agrupando e somando dados

    sql
Copiar código
SELECT nome, COUNT(*) AS total_pedidos
FROM Clientes
JOIN Pedidos ON Clientes.id_cliente = Pedidos.id_cliente
GROUP BY nome;

    Explicação: Recupera o nome de cada cliente e o número total de pedidos feitos por cada um, agrupados pelo nome do cliente.

  5. Ordenando os resultados

    sql
Copiar código
SELECT nome, data_nascimento
FROM Clientes
ORDER BY data_nascimento DESC;

    Explicação: Recupera o nome e data de nascimento de todos os clientes e ordena os resultados pela data de nascimento em ordem decrescente.

Resumo das Diferenças entre DDL, DML e SQL

Comando

Objetivo

Exemplo

DDL (Data Definition Language)

Define a estrutura de dados (tabelas, índices, etc.)

CREATE TABLE, ALTER TABLE, DROP TABLE

DML (Data Manipulation Language)

Manipula dados dentro das tabelas (inserir, atualizar, excluir)

INSERT INTO, UPDATE, DELETE

SQL (Structured Query Language)

Consulta dados para análise e relatório (recuperar dados)

SELECT, JOIN, GROUP BY, ORDER BY

Conclusão

  • DDL é usada para definir e modificar a estrutura do banco de dados.

  • DML permite a manipulação dos dados, incluindo inserção, atualização e exclusão.

  • SQL é a linguagem de consulta usada para buscar, combinar e analisar dados, sendo um subconjunto da DML.

Essas linguagens são fundamentais para o gerenciamento e manipulação de dados em bancos de dados relacionais, e o conhecimento de suas funcionalidades é essencial para quem trabalha com análise de dados ou desenvolvimento de sistemas de informação.
